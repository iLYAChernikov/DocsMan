роль
	объединение, символическая фильтрация для пользователей
	но ВСЕГДА ЕСТЬ 2 роли (user, superAdmin) (создаются автоматически и НЕ удаляются)
	для управления системой супер_админом
	также для отображения других элементов для других ролей

user
	только логин и пароль
	содержит несколько ролей

роль-user
	связь таблиц

профиль
	основная информация о пользователе

группа
	также для объединения пользователей и их фильтрации

профиль-группа
	связь таблиц
	+ устанавливает админов группы

уведомление
	уведомления о изменениях файлов, новости системы и т.п.

профиль-уведомление
	связь таблиц
	+ показывает статус: прочитано или нет

документ
	"образ" файла с именем и описанием
	также показывает статус: удалён или нет
		когда удалён не отображается но храниться в корзине

профиль-документ
	связь таблиц

загруженный файл
	показывает путь к физ файлу

история документа
	показывает состояния документа
	содержит
		ссылку на физ файлы прошлых версий
		время изменения
		описание

персональный документ
	содержит текст, ссылку на файл, тип документа

тип персонального документа
	автоматически создаётся тип - фото профиля

папка
	объединяет документы
	также содержит другие папки

папка-документ
	связь таблиц

папка-папка
	связь таблиц

профиль-папка
	связь таблиц
